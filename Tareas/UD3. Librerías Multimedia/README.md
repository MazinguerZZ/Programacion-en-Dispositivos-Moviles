# ACTIVIDAD 3.1 Desarrollo de aplicación Multimedia

**Ciclo Formativo de Grado Superior**  
**Desarrollo de Aplicaciones Multiplataforma**  
**Programación Multimedia y Dispositivos Móviles**

---

## 1. Objetivo

Demostrar conocimientos sobre desarrollo de aplicaciones para dispositivos móviles, considerando los siguientes aspectos:

- Analizar entornos de desarrollo multimedia (RA3.a).
- Reconocer las clases que permiten la captura, procesamiento y almacenamiento de datos multimedia (RA3.b).
- Utilizar clases para la conversión de datos multimedia de un formato a otro (RA3.c).
- Utilizar clases para procesar datos multimedia (RA3.d).
- Utilizar clases para el control de eventos, tipos de media y excepciones, entre otros (RA3.e).
- Utilizar clases para la creación y control de animaciones (RA3.f).
- Utilizar clases para reproducir contenidos multimedia (RA3.g).
- Depurar y documentar los programas desarrollados (RA3.h).

---

## 2. Enunciado

En parejas, realizad la siguiente tarea competencial evaluable para demostrar un conocimiento básico en relación con **"Librerías multimedia"**:

1. Cread una aplicación, tomando como referencia los contenidos vistos en clase, que permita realizar las siguientes actividades:

   - **a)** Captura de audio a través del micrófono del teléfono, permitiendo al usuario iniciar y terminar la grabación, y generando un fichero de audio.
   - **b)** Reproducción del fichero de audio generado cuando el usuario lo requiera.
   - **c)** Captura de imagen en formato JPEG usando una de las cámaras del dispositivo, mediante un intent u otra solución.
   - **d)** Mostrado de la imagen capturada en formato PNG.
   - **e)** Captura de vídeo con audio usando una de las cámaras del dispositivo, permitiendo al usuario iniciar y terminar la grabación, y generando un fichero de vídeo.
   - **f)** Reproducción del fichero de vídeo generado cuando el usuario lo requiera.
   - **g)** Animación de alguno de los componentes de la aplicación.

2. La aplicación debe ser personalizada en cuanto a colores e iconos de los componentes.

3. Desplegad la aplicación en un dispositivo móvil Android y probad su correcta ejecución, documentando en un fichero de texto las capturas extraídas del dispositivo móvil en las que se verifica su correcto funcionamiento.

4. Haced un `.zip` con el proyecto de Android Studio y el documento de texto y subidlo al aula virtual con el siguiente nombre:  
   `DAM2_PMM_UT3_Actividad1_Nombres-alumnos`

---

## 3. Evaluación

- La evaluación se realizará conforme a la siguiente lista de cotejo.
- La presentación de trabajos **no genuinos o plagiados** conllevará una calificación de **0 puntos** en la actividad para todos los alumnos involucrados.
- La actividad deberá entregarse antes de la fecha de cierre configurada en el aula virtual para ser evaluada. De no ser así, se puntuará con un 0. Se reabrirá el plazo de entrega para las convocatorias de Ordinaria y Extraordinaria para aquellos alumnos que deban presentarse a ellas.
- Tras la entrega, el docente podrá realizar preguntas a los alumnos para evaluar su grado de contribución a la actividad.
