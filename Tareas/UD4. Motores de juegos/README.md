# ACTIVIDAD 4.1 Análisis técnico y funcional de un videojuego real

**Ciclo Formativo de Grado Superior**  
**Desarrollo de Aplicaciones Multiplataforma**  
**Programación Multimedia y Dispositivos Móviles**

---

## 1. Objetivo

Demostrar conocimientos sobre selección y prueba de motores de juegos, considerando los siguientes aspectos:

- Identificar los elementos que componen la arquitectura de un juego 2D y 3D (RA4.a).
- Analizar los componentes de un motor de juegos (RA4.b).
- Analizar entornos de desarrollo de juegos (RA4.c).
- Analizar diferentes motores de juegos, sus características y funcionalidades (RA4.d).
- Identificar los bloques funcionales de un juego existente (RA4.e).
- Reconocer la representación lógica y espacial de una escena gráfica sobre un juego existente (RA4.f).

---

## 2. Enunciado

En parejas, realizad la siguiente tarea competencial evaluable para demostrar un conocimiento básico en relación con **"Motores de juegos"**:

1. Identificad un videojuego real de los últimos 5 años, para entornos móviles o de otra tipología a discreción de los alumnos.

2. Elaborad un informe o presentación donde se analice el videojuego seleccionado siguiendo los siguientes apartados:

### a) Arquitectura del juego

Identificando en la medida de lo posible los elementos de la arquitectura general:

- Sistema de input
- Bucle principal
- Lógica del juego
- Renderizado
- Salida audiovisual

### b) Componentes del motor de juegos

Detallando qué componentes del motor podéis encontrar y relacionándolos con su función en el juego, incluyendo:

- Motor gráfico
- Motor físico
- Sistema de audio
- Sistema de escenas
- Scripts y componentes

### c) Entorno de desarrollo utilizado

Describiendo:

- Motor de juegos empleado
- Editor visual
- IDE asociado
- Lenguaje de programación

### d) Análisis del motor y sus características

Analizando:

- Tipo de motor (2D / 3D)
- Ventajas del motor aprovechadas para este juego
- Posibles limitaciones
- Por qué lo consideráis adecuado para el proyecto

### e) Identificación de bloques funcionales

Describiendo:

- Jugador(es)
- Enemigos
- Escenario
- Interfaz (HUD)
- Sonido y música

### f) Representación lógica y espacial de la escena

Analizando:

- Coordenadas (X,Y o X,Y,Z)
- Cámara
- Capas o profundidad
- Colisiones

3. Subid el documento con el informe/presentación al aula virtual con el siguiente nombre:  
   `DAM2_PMM_UT4_Actividad1_Nombres-alumnos`
